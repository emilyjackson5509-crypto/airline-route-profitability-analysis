# airline-route-profitability-analysis
Multi-decade analysis (1993–2023) of U.S. airline route profitability, including revenue engineering, decile segmentation, Pareto analysis, and seasonal trend evaluation using Python and Tableau.

Dataset Source:
U.S. Airline Flight Routes and Fares (1993–2024), available on Kaggle:
 https://www.kaggle.com/datasets/bhavikjikadara/us-airline-flight-routes-and-fares-1993-2024
